# What is HTML ?

HTML: Hyper Text Markup Language used for build web structure using tags

CSS: language used to style and layout web pages by controlling the presentation of HTML elements.

JavaScript: used for creating interactive web pages. It runs in the browser and can manipulate HTML and CSS.

The code is excexuted  from top to bottom.
