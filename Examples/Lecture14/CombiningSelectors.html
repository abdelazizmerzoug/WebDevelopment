This is a highlighted div.

This paragraph is big and italicized.

This is a highlighted main point paragraph.

This article is a direct child of the section and will be green.

This paragraph is inside an article and will be red.

This paragraph is big and italicized inside an article and will be red.

            
This is the Css Style

                /* all classes with name Highlight */
                .highlight {
                    background-color: blue;
                }
            
                /* Every Element with class = "name_class" */
                p.big {
                    font-style: italic;
                }
            
                /* Every element with class highlight as well class main point */
                .highlight.mainpoint {
                    color: red;
                    background-color: black;
                }
            
                /* child selector (direct child) */
                section>article {
                    color: green;
                }
            
                /* Desendant selector any element inside first element  */
                article p {
                    color: red;
                }
        
            
This html code
